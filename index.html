BEAR.Ace

BEAR.AceはAceオンラインエディターをPHPから使いやすくしたユーティリティーです。オンラインエディタを簡単に設置することができます。またxdebugの機能を使ってエラーをAceへのリンクで表示したり、Syntaxエラーを検知するとそのAceエディタがそのURLまま開き、編集することができます。保存すると再読み込みが行われ、ケアレスミスの修正を最速で行えます。

$ cd BEAR.Ace
$ git fetch origin
$ git checkout gh-pages
